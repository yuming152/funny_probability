# 邮数工坊 - 大模型赋能教学练用一体化互动平台

## 项目简介

邮数工坊是一个专注于概率论与数理统计学科的教学辅助平台，通过大模型赋能，实现"教师端-学生端"双端场景的"教-学-练-用"全链路闭环。平台以学术严谨性与操作便捷性为核心，为教师和学生提供专业的概率统计教学解决方案。

## 项目结构

```
frontend/
├── index.html                 # 首页
├── teacher.html               # 教师端
├── student.html               # 学生端
├── bernoulli-experiments.html # 伯努利型实验
├── classical-experiments.html # 古典概型实验
├── conditional-experiments.html # 条件概率实验
├── distribution-experiments.html # 分布实验
├── geometric-experiments.html # 几何概型实验
├── independence-experiments.html # 独立性实验
├── limit-experiments.html     # 极限实验
├── multivariate-experiments.html # 多元统计实验
├── experiment-library.html    # 实验库
├── student-achievements.html  # 学生成就
├── student-ai-assistant.html  # 学生AI助手
├── student-exercise.html      # 学生练习
├── student-experiment.html    # 学生实验
├── student-review.html        # 学生复习
└── student-settings.html      # 学生设置
```

## 核心功能

### 教师端
- **教学决策中枢**：提供教学数据分析和决策支持
- **概率实验管理**：创建、编辑和管理各种概率统计实验
- **学情分析报告**：生成详细的学生学习情况分析报告
- **AI备课助手**：利用大模型辅助教师备课和教学设计

### 学生端
- **学习路径引导**：个性化学习路径推荐和引导
- **概率实验操作**：交互式概率实验操作和模拟
- **AI学习助手**：智能答疑和学习指导
- **协作交互设计**：学生之间的协作学习功能

### 技术特性
- **概率公式编辑器**：支持复杂概率公式的编辑和展示
- **统计图表工具**：多种统计图表的生成和分析
- **响应速度优化**：确保平台运行流畅
- **安全权限管控**：完善的用户权限管理

## 设计核心原则

1. **双端适配，链路闭环**：围绕"教师端-学生端"双端场景，以"教-学-练-用"全链路为核心逻辑，确保功能模块对应、数据实时联动

2. **分层可视化，聚焦核心场景**：针对"概率论与数理统计"学科特性，采用"基础功能+进阶工具+创新实验"的分层视觉设计，通过色彩分区、模块优先级排序，让核心功能一目了然

3. **AI赋能隐形化，降低操作门槛**：将大模型能力融入各环节，以"轻交互"形式呈现，避免复杂操作流程。同时，通过动态演示降低抽象知识理解难度，符合教育产品"辅助而非替代"的设计逻辑

## 技术栈

- **前端框架**：HTML5, CSS3, JavaScript
- **UI框架**：Tailwind CSS
- **图标库**：Font Awesome
- **图表库**：Chart.js
- **数学公式**：MathJax, KaTeX

## 快速开始

1. **克隆项目**
   ```bash
   git clone <repository-url>
   ```

2. **进入项目目录**
   ```bash
   cd funny_probability
   ```

3. **启动本地服务器**
   - 使用VS Code的Live Server扩展
   - 或使用Python内置服务器：
     ```bash
     python -m http.server 8000
     ```
   - 或使用Node.js的http-server：
     ```bash
     npx http-server -p 8000
     ```

4. **访问平台**
   在浏览器中打开 `http://localhost:8000/frontend/index.html`
