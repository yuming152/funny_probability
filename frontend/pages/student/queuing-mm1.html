理论分析

M/M/1排队模型的基本假设

  • 到达过程：泊松过程，到达率为 λ
  • 服务时间：指数分布，服务率为 μ
  • 服务台数量：1个
  • 排队规则：先到先服务（FCFS）
  • 系统容量：无限
  • 顾客源：无限

性能指标计算公式

系统利用率： ρ = λ / μ

稳态队列长度： Lq = ρ² / (1 - ρ)

稳态系统中的顾客数： Ls = ρ / (1 - ρ)

平均等待时间： Wq = Lq / λ

平均逗留时间： Ws = Wq + 1/μ

稳态概率分布

P₀ = 1 - ρ （系统中没有顾客的概率）

Pₙ = ρⁿ(1 - ρ) （系统中有n个顾客的概率）
