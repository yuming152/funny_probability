理论解释

随机区间生成

在 [0,1] 区间上生成随机区间的方法是：

  • 生成两个随机数 a 和 b，其中 0 ≤ a, b ≤ 1
  • 区间的起点为 min(a, b)，终点为 max(a, b)
  • 区间长度为 |a - b|

区间相交判断

两个区间 [x1, y1] 和 [x2, y2] 相交的充要条件是：

  • x1 < y2 且 x2 < y1
  • 或者等价于：max(x1, x2) < min(y1, y2)

概率计算

对于两个在 [0,1] 上随机生成的区间，它们相交的概率可以通过几何概率计算：

  • 设两个区间的端点分别为 (a, b) 和 (c, d)，其中 a < b, c < d
  • 所有可能的端点组合构成一个 4 维空间 [0,1]^4
  • 相交的条件是 max(a, c) < min(b, d)
  • 通过积分计算，两个随机区间相交的概率为 1/3

推广到多个区间

对于 n 个在 [0,1] 上随机生成的区间，它们全部相交的概率计算会更加复杂，通常需要通过模拟方法来估计。
